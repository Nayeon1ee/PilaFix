# | PILAFIX



## 🖥️ 프로젝트 소개
필라테스 수업을 운영하는 센터와 회원, 그리고 강사들 간의 효율적인 소통과 예약 시스템을 제공

통계 및 보고서를 통한 매출관리와 편리한 사이트 관리 서비스 제공 
<br>

## 🕰️ 개발 기간
* 23.11.08. - 23.12.08(1개월)

## ⚙️ 개발 환경
- `Java 11`
- `JDK 1.8.0`
- **IDE** : STS 3.9
- **Framework** : Springframework 5.3.27
- **Database** : PostGreSQL
- **ORM** : Mybatis
